Name: Ashley Espinoza  
Course: 2nd Semester  
Section: A  

This repository contains my solution for the lab "Reports with AWK in a Gradebook File."

## Tasks

- Task 1: Count submissions without the header line.
- Task 2: Count distinct students.
- Task 3: Print the FINAL assignment report.
- Task 4: Count failed submissions.
- Task 5: Generate a report per assignment with minimum, maximum, and average.
- Task 6: Calculate weighted percentage and letter grade for each student.
- Task 7: Run the final report from a Bash script and sort it by student name.
- Bonus: Print the class average row.
